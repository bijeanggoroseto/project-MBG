## Struct
1. Bahan: berisi nama bahan, jumlah stok bahan, dan nama supplier.
2. Masakan: berisi nama menu masakan, apa saja bahan yang digunakan, berapa jumlah tiap bahannya, dan langkah-langkah memasak.
3. User: berisi username dan password.
## Fitur
### 1. Login dan Registrasi
User bisa login menggunakan akun yang sudah ada atau registrasi dengan akun baru.
### 2. Atur Bahan Masakan1. 
- Input bahan masakan.
- Tambah jumlah bahan.
- Edit bahan.
- Hapus bahan.
- Tampilkan bahan masakan.
- Cari bahan masakan.
### 3. Atur Menu Masakan
- Input menu masakan.
- Hapus menu masakan.
- Tampilkan menu masakan.
- Cari  menu masakan.
- Masak masakan.
### 4. Menyimpan data ke dalam file
Saat user keluar dari aplikasi maka semua perubahan yang terjadi pada data akan otomatis tersimpan ke dalam file Bahan.txt, Masakan.txt dan User.txt.
